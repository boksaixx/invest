삼성전자 · SK하이닉스 · 한미반도체 · 삼성전기 · DB하이텍 — 반도체 5종목 **단기 매매 어드바이스** 인공지능 에이전트입니다.

- 📱 **토스처럼 쉬운 화면**: 현금과 매수가만 입력하면 끝
- 🔟 **0~10점 매수/매도 강도**: 미보유 종목은 "지금 얼마나 강하게 사야 하는지", 보유 종목은 "얼마나 강하게 팔아야 하는지"를 단일 점수로 표시. 5종목을 강도순으로 랭킹해 화면 맨 위에서 바로 확인 가능
- 🤖 **AI 이중 분석**: 룰 엔진(기술적+장중 분석) + Claude(종합 판단, 점수까지 최종 검증) + Gemini(실시간 속보 수집)
- 📊 **장중(분봉) 데이터**: VWAP, 갭, 오프닝레인지 브레이크아웃, 당일 모멘텀 — "지금 이 순간" 판단 근거 (필요할 때만 펼쳐보는 상세 화면으로 분리)
- 🌡️ **매크로 확장**: VIX(공포지수), S&P500·나스닥100 선물, CNN 공포탐욕지수까지 반영
- 🎯 **실행 가능한 플랜**: "관망하세요"로 끝나지 않고 진입 트리거·분할 매매·무효화 조건을 항상 제시
- ⏰ **자동 수집(비용 최적화)**: 장중엔 15분 간격, 장 마감~아침 7시엔 호출 없음, 주말은 일요일 1회만 — 월 예상 비용 $10 이내
- 🛡️ **리스크 관리 내장**: 손절가 자동 계산, 1회 손실 1% 제한, 물타기 금지, 손익비 1:2 원칙, 섹터 집중도 경고
- 🔒 **비밀번호 보호**: 배포 주소를 아는 타인이 API 크레딧을 소모하지 못하도록 간단 비밀번호 게이트 내장
- 📈 **백테스트 참고 통계**: 3개년 일봉 기준 과거 유사 신호의 승률·평균수익률을 매주 갱신해 참고용으로 제공

> ⚠️ 본 서비스는 투자 참고 정보이며 수익을 보장하지 않습니다. 모든 투자의 최종 판단과 책임은 본인에게 있습니다.
> 시세는 무료 공개 API 기반이라 최대 15~20분 지연될 수 있습니다 — 실제 주문 전 증권사 앱에서 최신가를 반드시 재확인하세요.

## 🚀 설치하기 (버튼만 누르면 됩니다 — 약 15분)

### 준비물: API 키 2개 만들기

**① Claude API 키** (AI 분석 담당, 유료 — 사용량만큼 과금)
1. https://console.anthropic.com 접속 → 구글 계정으로 가입/로그인
2. 왼쪽 메뉴 **API Keys** → **Create Key** 버튼 클릭
3. 나온 `sk-ant-...` 로 시작하는 긴 문자를 복사해서 메모장에 저장
4. **Billing** 메뉴에서 결제 카드 등록 (월 $5~10 정도면 충분합니다)
> ⚠️ claude.ai 유료 구독(Pro/Max)과 API 결제는 **완전히 별개**입니다. claude.ai 구독이 있어도 API는 별도로 크레딧을 충전해야 동작합니다.

**② Gemini API 키** (뉴스 수집 담당 — 구글 검색 그라운딩 사용)
1. https://aistudio.google.com/apikey 접속 → 구글 계정 로그인
2. **API 키 만들기** 버튼 클릭 (기존 키가 있으면 그대로 사용해도 됩니다)
3. 나온 `AIza...` 로 시작하는 문자를 복사해서 메모장에 저장
4. 만든 키가 어느 **Google Cloud 프로젝트**에 연결됐는지 확인 → https://console.cloud.google.com/billing 에서 그 프로젝트에 **결제 계정(Billing Account)이 연결되어 있는지** 확인. 연결이 안 돼 있으면 "결제 계정 연결" 버튼으로 카드 등록
5. https://aistudio.google.com 좌측 하단 **Plan/사용량** 메뉴에서 무료 등급(Free tier)이 아니라 종량제(Pay-as-you-go / Paid tier)로 표시되는지 확인

> ⚠️ **자주 헷갈리는 3가지가 실은 전부 다른 것입니다:**
> 1. **구글 원(Google One)의 제미나이 어드밴스드(Gemini Advanced) 구독** — 챗봇 앱 gemini.google.com을 쓰는 개인 구독입니다. 이 앱을 아무리 유료로 써도 **API 키 사용량과는 완전히 무관**합니다.
> 2. **API 키 자체의 유·무료 등급** — API 키는 무료로 "발급"되지만, 그 키가 연결된 Google Cloud 프로젝트에 **결제 계정이 연결·활성화**되어 있어야 비로소 더 넉넉한 유료 등급 한도로 호출됩니다. 위 4번 단계로 확인/등록하세요.
> 3. **구글 검색 그라운딩(google_search) 전용 쿼터** — 이 에이전트는 실시간 속보를 가져오기 위해 Gemini의 "구글 검색" 기능을 함께 씁니다. 이 기능은 일반 텍스트 생성과 **별도의, 더 빡빡한 자체 쿼터**를 씁니다. 즉 프로젝트가 유료 등급이어도 검색 그라운딩 호출만 429(쿼터 초과) 에러가 날 수 있습니다 — 텍스트 호출만 하는 자가진단(`/api/health`)이 "정상"으로 나와도 실제 뉴스 수집은 실패하는 경우가 바로 이 때문입니다. 반복적으로 429가 발생하면 Google Cloud 콘솔에서 해당 프로젝트의 "Generative Language API" 쿼터/한도 상향을 요청하세요.
>
> 이 에이전트는 429/401/403 오류가 뜨면 같은 요청을 반복해서 쿼터를 낭비하지 않도록 즉시 재시도를 중단하고 결과를 잠깐 캐시합니다(정상적인 보호 장치이며, 유료 등급 여부와 무관하게 항상 켜져 있습니다).

### 1단계. GitHub 설정 (자동 수집 에이전트 켜기)

1. 이 저장소(GitHub) 페이지 상단의 **Settings** 클릭
2. 왼쪽 메뉴에서 **Secrets and variables → Actions** 클릭
3. **New repository secret** 버튼을 눌러 아래 2개를 각각 추가:
   - Name: `ANTHROPIC_API_KEY` / Secret: 아까 저장한 Claude 키 붙여넣기 → **Add secret**
   - Name: `GEMINI_API_KEY` / Secret: 아까 저장한 Gemini 키 붙여넣기 → **Add secret**
4. 저장소 상단 **Actions** 탭 클릭 → 초록색 **"I understand... enable them"** 버튼이 보이면 클릭
5. **Settings → Actions → General**에서 "Allow all actions and reusable workflows" + "Read and write permissions"가 선택되어 있는지 확인 (자동수집 데이터를 저장소에 커밋하려면 필요)
6. 왼쪽에서 **Trading Agent** 클릭 → 오른쪽 **Run workflow** 버튼 → 초록색 **Run workflow** 클릭 (첫 실행 테스트)

✅ 이제 평일 장중(09:00~15:30)엔 15분마다, 개장 전(08:30)·마감 후(16:10)엔 브리핑이 자동으로 돌아갑니다.
**밤 11시~아침 7시(KST)는 호출하지 않고, 주말은 일요일 저녁 1회만** 점검(+ 데이터 최신화)합니다 — 잠자는 시간에 API 비용이 새는 것을 막기 위한 설계입니다.
첫 실행 때 3개년 과거 데이터(주가/환율/해외지수)와 백테스트 통계도 자동으로 채워지고, 이후 매주 일요일마다 갱신됩니다.

### 2단계. Vercel 배포 (내 전용 앱 만들기)

1. https://vercel.com 접속 → **Sign Up** → **Continue with GitHub** 로 가입
2. **Add New... → Project** 클릭
3. 목록에서 **invest** 저장소 옆의 **Import** 클릭
4. **Environment Variables** 항목을 펼쳐서 아래 항목을 추가:
   - Key: `ANTHROPIC_API_KEY` / Value: Claude 키 붙여넣기
   - Key: `GEMINI_API_KEY` / Value: Gemini 키 붙여넣기
   - Key: `APP_PASSWORD` / Value: 원하는 비밀번호 (생략하면 기본 비밀번호 `0815`가 자동 적용됩니다. 더 안전한 값으로 바꾸고 싶으면 원하는 문자열로 설정하세요)
5. **Deploy** 버튼 클릭 → 1~2분 기다리면 완료 🎉
6. 완료 화면의 주소(예: `https://invest-xxx.vercel.app`)를 누르면 앱이 열립니다. 비밀번호 입력 화면이 먼저 나오며, `APP_PASSWORD`를 따로 설정하지 않았다면 **`0815`** 를 입력하면 됩니다. **휴대폰 홈 화면에 추가**해두면 앱처럼 쓸 수 있어요 (사파리/크롬 공유 버튼 → "홈 화면에 추가")
7. 배포 후 `주소/api/health`를 열어 모든 항목이 "정상"인지 자가진단할 수 있습니다.

> 🔒 **비밀번호 보호**: 정식 로그인/계정 시스템이 아니라, 배포 주소를 아는 타인이 함부로 들어와 내 Claude/Gemini API 크레딧을 소모하지 못하게 막는 최소한의 문 잠금입니다. 별도 설정 없이도 기본 비밀번호 **`0815`** 로 항상 켜져 있으며, Vercel 프로젝트의 **Settings → Environment Variables**에서 `APP_PASSWORD`를 추가/변경하면 그 값으로 바뀝니다(환경변수를 바꾼 뒤에는 **Deployments 탭에서 Redeploy**를 눌러야 적용됩니다). GitHub Actions의 자동 수집 스크립트는 이 비밀번호와 무관하게 계속 동작합니다(웹 화면만 보호 대상).

## 📱 사용법

1. 앱을 열고 오른쪽 위 **"내 자산 입력"** 을 눌러 보유 현금을 입력합니다 (기본 2,000만원)
2. 주식을 사면 그 종목의 **매수 평단가**와 **수량**을 입력합니다
3. **"지금 AI 정밀 분석 받기"** 버튼을 누르면:
   - 실시간 시세 + 일봉/장중 기술적 지표 + 실시간 뉴스·속보 + 환율/해외증시/VIX/선물/공포탐욕지수를 모두 종합해
   - 화면 맨 위 **"지금 뭘 해야 하나"** 랭킹에 5종목을 **매수/매도 강도(0~10점)** 순으로 정렬해 보여줍니다. 미보유 종목은 "지금 얼마나 강하게 사야 하는지", 보유 종목은 "얼마나 강하게 팔아야 하는지"를 한 눈에 파악할 수 있습니다
   - 각 종목 카드에도 같은 0~10점 게이지와 함께 **목표가·손절가**(얼마에 팔지/얼마에서 손절할지)를 바로 보여주고, VWAP·오프닝레인지·분할매매 같은 상세 지표는 "자세히 보기"를 눌러야 나오도록 감춰서 화면을 단순하게 유지합니다
4. 보유 정보를 입력해두면 "지금 팔아야 할지·얼마나 강하게"를, 미보유면 "지금 사야 할지·얼마나 강하게"를 점수로 알려줍니다

## ❓ 자주 묻는 질문

**Q. 자동으로 주식을 사고 파나요?**
아니요. 이 에이전트는 **조언만** 합니다. 실제 매매는 본인이 증권사 앱(MTS)에서 직접 합니다. 자동 매매는 증권사 API 인증·법적 요건이 필요해 의도적으로 제외했습니다.

**Q. 비용이 얼마나 드나요?**
GitHub Actions·Vercel은 무료 한도로 충분합니다. Gemini는 기본적으로 무료 등급 한도 안에서도 동작하도록 설계했지만, 구글 검색 그라운딩(속보 수집) 쿼터가 무료 등급에서는 빡빡해 자주 429 오류가 날 수 있습니다 — 안정적으로 쓰려면 위 "Gemini API 키" 섹션의 안내대로 프로젝트에 결제 계정을 연결해 유료 등급으로 올리는 것을 권장합니다(사용량 자체가 매우 적어 실제 청구액은 미미한 수준입니다). Claude API는 사용량 과금인데, 자동수집은 저렴한 모델(Haiku)로 장중 15분 간격(하루 약 27회) + 개장전/마감 브리핑만 돌고 밤 11시~아침 7시·평일 외에는 호출하지 않도록 설계해 자동수집만으로는 월 $2~4 수준입니다. 여기에 화면에서 직접 누르는 "AI 정밀 분석"(고급 모델) 호출 비용이 하루 몇 회 기준으로 더해져도, 합계는 **월 $10 이내**를 목표로 설계했습니다. 다만 이는 예상치이며 실제 사용 패턴에 따라 달라질 수 있으니, [console.anthropic.com/settings/billing](https://console.anthropic.com/settings/billing)에서 **월 지출 한도(Spending limit)를 직접 설정**해두는 것을 강력 권장합니다 — 앱 코드로는 API 자체의 과금을 막을 수 없기 때문입니다.

**Q. 시세가 실시간인가요?**
야후 파이낸스/네이버 금융 무료 API를 쓰기 때문에 **최대 15~20분 지연**될 수 있습니다. 화면에 시세 옆 "n분 전 시세"로 표시됩니다. 실제 주문 직전에는 반드시 증권사 앱에서 최신 호가를 확인하세요. 이 한계 때문에 진입/무효화 조건을 고정 가격이 아니라 "조건 충족 여부"로 설계했지만, 완전히 없앨 수는 없습니다.

**Q. 알림(카카오톡 등)은 없나요?**
현재는 없습니다. 자동수집은 장중 15분 간격으로 데이터만 쌓고, 실제 조언은 앱에서 버튼을 눌러야 받을 수 있습니다. 장중에는 주기적으로 앱을 열어 확인하는 것을 권장합니다.

**Q. 자동 수집 데이터는 어디에 쌓이나요?**
이 저장소의 `data/` 폴더에 자동 커밋됩니다. `data/latest.json`(최신), `data/log/날짜.json`(일별 기록), `data/market-history.json`(3개년 주가/환율/지수), `data/backtest.json`(백테스트 통계, 매주 갱신)입니다.

**Q. 앱 분석과 자동 수집의 차이는?**
자동 수집(장중 15분 간격)은 시장 전체 관점의 신호와 뉴스를 축적하고, 앱에서 버튼을 누르면 **내 보유 현황을 반영한** 맞춤 분석이 그 자리에서 실행됩니다.

**Q. 백테스트는 얼마나 믿을 수 있나요?**
룰 엔진의 일봉 기술적 점수가 68점 이상이었던 과거 시점을 찾아 5/10거래일 후 종가 수익률을 집계한 **단순 참고 통계**입니다. 장중 변동성·뉴스·매크로는 과거 시점으로 재현할 수 없어 제외했고, 실제 체결가·슬리피지·거래비용도 반영하지 않았습니다. "과거 승률 OO%"는 통계적 경향일 뿐 다음 매매의 결과를 보장하지 않습니다 — 화면과 AI 판단 모두 이 통계를 절대적 근거로 쓰지 않고 참고 지표로만 언급하도록 설계했습니다.

**Q. 코드를 수정하면 어떻게 반영되나요?**
GitHub에서 파일을 수정(커밋)하면 Vercel이 자동으로 다시 배포합니다. 별도 조작이 필요 없습니다.

## ⚠️ 알려진 한계 (정직하게 밝힙니다)

- **시세 지연**: 무료 API 특성상 최대 15~20분 지연 가능. 실시간 호가·체결 데이터가 아닙니다.
- **분석 소요 시간**: AI 정밀 분석은 30초~2분이 걸립니다. 그 사이 가격이 움직일 수 있어, 결과는 고정가가 아니라 조건부 트리거 중심으로 제시됩니다.
- **백테스트는 단순 재현치**: 3개년 일봉 종가만으로 재현한 참고 통계이며, 장중 변동성·뉴스·매크로·거래비용·슬리피지는 반영하지 않았습니다. 과거 승률이 미래 수익을 보장하지 않으며, 검증된 필승 전략이 아니라 확률적 우위를 노리는 참고 지표입니다.
- **공휴일 인식 제한**: 양력 고정 공휴일(신정·삼일절 등)만 자동 인식합니다. 설날·추석 등 음력 공휴일 주간은 별도로 확인하세요.
- **비밀번호는 정식 인증이 아님**: 기본값(`0815`) 또는 직접 설정한 `APP_PASSWORD` 값 하나로 여닫는 단일 비밀번호 게이트일 뿐, 계정별 권한 관리나 무차별 대입 공격 방어 같은 정식 인증 시스템이 아닙니다. 배포 주소를 다른 사람과 공유하지 말고, 가능하면 `APP_PASSWORD` 환경변수를 본인만 아는 값으로 바꾸는 것을 권장합니다.
- **공포탐욕지수**: 미국 시장(CNN) 기준 지표이며 비공식 데이터 소스라 수집이 실패할 수 있습니다(실패 시 조용히 생략).
